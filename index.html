LibreDTE®

¡Facturación electrónica libre para Chile!

Somos la primera solución de facturación electrónica libre para Chile. Con LibreDTE podrás administrar múltiples empresas desde una misma cuenta, autorizar a otros usuarios a trabajar con tu compañía e integrar fácilmente tu actual sistema computacional a nuestra API.

Servicios

Obtén más funcionalidades y soporte con SASCO SpA

Servicio Plus

Desde 1 UF +IVA/mes
¿Deseas usar www.libredte.cl?
Módulos facturación, contabilidad, inventario, RRHH, cobros masivos programados, pagos electrónicos, tienda electrónica entre otros.
¡Quiero ser Plus!

Servicios web SII

Valor 2 UF +IVA/mes
¿Recuperar datos desde el SII?
Accede a timbraje automático de folios, listado boletas de honorarios, registro de compra y venta, entre otros, desde tu aplicación.
¡Acceder a la API!

Certificación DTE

Desde 4 UF +IVA
¿No puedes certificar tu software?
Realizamos el proceso de certificación ante el SII para que seas autorizado a emitir DTE, luego, podrás usar tu software para la emisión.
¡Certificar mis DTE!

Instalación local

Desde 35 UF +IVA
¿LibreDTE en tu servidor?
Instalamos en tu propio servidor la aplicación web de LibreDTE para que puedas usar el módulo de facturación y emitir todos tus documentos.
¡Instalar LibreDTE!

Servicios web

Se provee de una API para interactuar con la aplicación web de LibreDTE. Tiene la ventaja de no requerir instalar la aplicación ni biblioteca.

Documentación de la API y SDKs para conectar a la API

Usando el cliente

Este cliente permitirá consumir de manera sencilla los servicios web de LibreDTE. Podrás hacer esto a través de llamadas a "system" o similar, como si fuese cualquier programa de línea de comandos.

Descargar el cliente

Directo con PHP

Si la aplicación está escrita en PHP y está publicado su código fuente bajo licencia AGPL, es posible utilizar la biblioteca de LibreDTE, se debe incluir usando composer y se podrá empezar a facturar.

composer require sasco/libredte
