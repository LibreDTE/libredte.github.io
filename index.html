¿Necesitas certificar una empresa?

Realizamos el proceso de certificación del set básico(1) por 4 UF(2)

Si deseas utilizar facturación electrónica, ya sea con LibreDTE u otro proveedor. Pero no sabes cómo hacer el proceso de certificación ante el SII. Nosotros podemos realizarlo por ti. El costo de certificar el set básico1 es de 4 UF(2). Si necesitas certificar otros sets consúltanos. Revisa los aquí los requisitos para hacer la certificación.

  1. Set básico corresponde a:
    • Factura electrónica (33)
    • Nota de débito electrónica (56)
    • Nota de crédito electrónica (61)
  2. Costo en valor neto y al contado. Costos adicionales por otros tipos de DTE a certificar.

Integración

Webhooks

La manera más simple de integrar, ya tenemos todo listo. Sólo debes configurar tu aplicación para que use nuestros webhooks.

¡Solicita gratis un nuevo webhook para la aplicación que uses!

Servicios web

Se provee de una API para interactuar con la aplicación web de LibreDTE. Tiene la ventaja de no requerir instalar la aplicación ni biblioteca.

Documentación de la API y SDKs para conectar a la API

Usando el cliente

Este cliente permitirá consumir de manera sencilla los servicios web de LibreDTE. Podrás hacer esto a través de llamadas a "system" o similar, como si fuese cualquier programa de línea de comandos.

Descargar el cliente

Directo con PHP

Si la aplicación está escrita en PHP y está publicado su código fuente bajo licencia AGPL, es posible utilizar la biblioteca de LibreDTE, se debe incluir usando composer y se podrá empezar a facturar.

composer require sasco/libredte

Sobre nosotros

Soporte oficial por SASCO SpA

SASCO SpA es la empresa que está detrás del proyecto LibreDTE, somos una empresa pequeña, formada por profesionales con ganas de ser un aporte para la sociedad.
