Usando el Cliente

Este cliente permitirá consumir de manera sencilla los servicios web de LibreDTE. Podrás hacer esto a través de llamadas a "system" o similar, como si fuese cualquier programa de línea de comandos.

Descargar el Cliente de Terminal

Directo con PHP

Si la aplicación está escrita en PHP puedes instalar con composer nuestra biblioteca. Pero recuerda, el código fuente de tu software debe estar publicado bajo licencia AGPL.

composer require sasco/libredte

Soporte oficial por SASCO SpA

SASCO SpA es la empresa que está detrás del proyecto LibreDTE, somos una PYME que trabaja por las PYMEs. Formada por profesionales con ganas de ser un aporte para la sociedad.

Diseñamos y mejoramos nuestra solución de acuerdo a la experiencia en desarrollo de aplicaciones y facturación electrónica. Somos la empresa que provee el soporte oficial a LibreDTE. Somos expertos en LibreDTE.
